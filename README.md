DESCRIPTION : This Python program retrieves a 5-day weather forecast for Bengaluru using the OpenWeatherMap API.It sends an HTTP request with the city name, API key, and metric units.The received JSON data is processed and cleaned for analysis.Relevant weather parameters such as temperature, humidity, and pressure are extracted.UNIX timestamps are converted into a readable date and time format.The processed data is stored in a pandas DataFrame.Seaborn and Matplotlib libraries are used for data visualization.Line graphs are created to show temperature trends over time.Additional graphs display humidity and pressure variations.
Overall, the program helps users analyze and understand weather patterns effectively.
